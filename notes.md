# Regular verbs

## Presente
| -ar (falar)   | -er (comer)   | -ir (abrir)   |
| ---           | ---           | ---           |
| fal`ar`       | com`o`        | abr`o`        |
| fal`as`       | com`es`       | abr`es`       |
| fal`a`        | com`e`        | abr`e`        |
| fal`amos`     | com`emos`     | abr`amos`     |
| fal`aram`     | com`em`       | abr`em`       |

## Pretérito perfeito
| -ar (falar)   | -er (comer)   | -ir (abrir)   |
| ---           | ---           | ---           |
| fal`ei`       | com`i`        | abr`i`        |
| fal`aste`     | com`este`     | abr`iste`     |
| fal`ou`       | com`eu`       | abr`iu`       |
| fal`ámos`     | com`emos`     | abr`imos`     |
| fal`aram`     | com`eram`     | abr`iram`     |
